Personal Portfolio Website

A modern, responsive personal portfolio website showcasing my work as a Software Engineer specializing in AI/ML, DevOps, and Web Development.

🔑 Key Features

  • Dynamic Content Loading

    Single-page application feel with seamless project/blog navigation without page reload.

  • Interactive Code Examples

    Copy-to-clipboard for code snippets, syntax highlighting, and interactive demos.

  • Blog Integration

    Technical blog posts with proper formatting, responsive layout, and syntax highlighting.

  • Responsive Design

    Mobile-first design, smooth transitions, professional typography, and modern color scheme.

📖 Technologies Used

Frontend: HTML5, CSS3, JavaScript (Vanilla)
Styling: CSS custom properties, responsive design
Icons: Ionicons
Fonts: Google Fonts (Poppins)
Deployment: Static hosting ready

🚀 Getting Started

# Clone the repository
git clone https://github.com/santhoshclx/portfolio.git
cd portfolio

# Open locally
# Open index.html in your browser, or use a local server:

# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using VS Code Live Server extension


View the portfolio at http://localhost:8000 (if using local server) or by opening index.html directly.

📱 Sections

  • About – Personal introduction and services offered
  • Resume – Professional experience and education
  • Projects – Showcase of technical projects with interactive demos
  • Blog – Technical articles and tutorials
  • Contact – Contact form and social media links

🎨 Customization

Content

  • Update personal information in index.html
  • Add new projects in the projects/ directory
  • Add new blog posts in the blogs/ directory
  • Update project/blog links in the main index.html file

Colors

The website uses CSS custom properties for easy theming. Main colors can be modified in assets/css/style.css:

:root {
  --orange-yellow-crayola: #ffdb70;
  --smoky-black: #0c0c0c;
  --eerie-black-1: #1e1e1f;
  /* ... other color variables */
}
